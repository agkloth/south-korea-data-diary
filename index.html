Pyeongtaek is home to the world's largest US military base outside America, Korea's #1 car export port, and the world's biggest Samsung semiconductor plant.

This project uses open Korean government data to analyze a fast-growing city that almost no one is studying. Every chart is interactive, reproducible, and built with free open-source tools.

Methodology How this project works
01
Data sources
Korean government open data portals — KOSIS, AirKorea API, GPPC port authority, Pyeongtaek city portal. All free.
02
Python + Plotly
Charts built in Python using Plotly — the same library used by professional data teams. Fully interactive HTML output.
03
GitHub Actions
Automated workflow generates charts on demand. One dropdown pick — chart appears in 60 seconds, committed to the repo.
04
GitHub Pages
Every generated chart is automatically published as a live URL. No server, no hosting costs, no maintenance.
05
LinkedIn
Each chart comes with an auto-generated caption draft. Screenshot + caption + live link = one post.
